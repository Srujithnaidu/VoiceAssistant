## ðŸ“ŒInstallation

   ```bash

   ```

3. **Create and Activate Virtual Environment**
   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   ```

4. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```
